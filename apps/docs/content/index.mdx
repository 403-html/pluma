# Welcome to Pluma Docs

Pluma is a self-hosted feature flag system. Manage flags via a web UI and evaluate them in your application with a lightweight SDK.

## Getting Started

- **Admin UI** – manage feature flags, environments, and SDK tokens through the web interface.
- **SDK** – evaluate flags in your application using `@pluma-flags/sdk`.
