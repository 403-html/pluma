# Welcome to Pluma

**Pluma** is a self-hosted feature flag system. Manage flags via a web UI and evaluate them in your application with a lightweight SDK.

> *Pluma (Spanish: feather) — lightweight by design.*

## What is Pluma?

Pluma lets you control feature rollouts without redeploying your application. You define flags in the UI, and your application evaluates them at runtime using the SDK.

Key capabilities:

- **Feature flags** — enable or disable features per environment
- **Per-subject targeting** — evaluate flags for individual users or entities
- **Lightweight SDK** — minimal footprint, works with any Node.js or React app
- **Self-hosted** — your data stays on your infrastructure
