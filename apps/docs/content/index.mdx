# Pluma Documentation

Pluma is a self-hosted feature flag system. Manage flags via a web UI and
evaluate them in your application with a lightweight SDK.

## Features

- **Web UI** — create and manage flags, environments, projects, and SDK tokens.
- **Lightweight SDK** — fetches a flag snapshot once and evaluates locally; no
  round-trip per flag check.
- **Per-subject targeting** — allow/deny lists and deterministic rollout
  percentages.
- **Parent/child flag hierarchies** — organise flags into trees with inherited
  evaluation.
- **Self-hosted via Docker Compose** — no external dependencies beyond
  PostgreSQL.
